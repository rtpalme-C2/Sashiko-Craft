App edits vs. file updates Changes you make directly in the app save to Google Drive automatically — no GitHub upload needed. When Claude updates the HTML file, you need to upload it to GitHub and tap Reset local data so the app reloads the new seed data.
Quick reference
Mark an item delivered Edit button on the card → change Status → Save
Add a new item + Add Item → fill in fields → Save
Add items from an invoice Upload to Claude → download updated file → upload to GitHub → Reset local data
Find something Search bar or category/status filters
Fix a mistake Edit button on the card → make changes → Save
Remove an item Delete button on the card → confirm
